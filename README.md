# Infinite Journey Reiki Healing Website 🌿

A modern Reiki healing website designed to provide online consultations, healing courses, and spiritual guidance. This platform helps Reiki practitioners connect with clients, showcase services, and manage leads through CRM integration.

## 🚀 Features

* Responsive and modern UI
* Reiki healing service pages
* Online consultation booking
* Course and learning sections
* Testimonials and client feedback
* Lead capture for CRM funnel integration
* SEO-friendly structure

## 🛠 Tech Stack

* **Frontend:** React, Vite, HTML, CSS, JavaScript
* **Backend:** Node.js
* **Deployment:** GitHub Pages
* **Integration:** CRM / Funnel system

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/KartheekNelloree/InfiniteJourneyReikiHealingWebsite.git
```

Navigate to the project:

```
cd InfiniteJourneyReikiHealingWebsite/frontend
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

## 🌐 Deployment

Build the project:

```
npm run build
```

Deploy to GitHub Pages:

```
npm run deploy
```

Live Website:

https://kartheeknelloree.github.io/InfiniteJourneyReikiHealingWebsite/
