# Infinite Journey Reiki Healing Center

A modern, responsive website for Geetha — Reiki Growth Coach, built with React + Vite + Tailwind CSS.

## Tech Stack

- **Frontend:** React 19, Vite 7, Tailwind CSS 4
- **Animations:** Framer Motion
- **Routing:** React Router DOM
- **Deployment:** GitHub Pages + Vercel

## Features

- Home, About, Services, Courses, Booking, Testimonials, Contact pages
- Appointment booking form
- Responsive design with mobile navigation
- Infinity symbol branding throughout
